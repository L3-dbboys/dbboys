功能特性

覆盖数据库全生命周期

从连接、开发、管理到卸载，提供一站式的数据库桌面工具。本地客户端形态运行，连接信息和运行数据保留在本地。

🔌

连接管理

按分类管理连接、支持多种 JDBC 驱动切换、GBase 8s / Informix 组名连接、只读模式、断线重连与长连接保活。

📝

SQL 工作台

多标签页编辑器，语法高亮、自动补全、格式化、绑定变量、执行计划、自动/手动提交模式与变更历史追踪。

📊

结果集 & 数据导入导出

分页加载、在线编辑（支持主键/rowid）、LOB 查看编辑；导出到 CSV / JSON / SQL，DDL 批量生成。

🌳

对象管理

浏览数据库/模式、表、视图、索引、序列、函数、存储过程、触发器、包、用户等对象，图形化建表改表，DDL 导出。

🏠

实例运维

一键巡检、运行日志、空间容量图表、参数管理、锁表处理与 Kill 会话、实例启停。按数据库能力动态展示功能。

🚀

远程安装卸载

通过 SSH 远程安装/卸载 GBase 8s、Informix、MySQL。向导式流程，自动处理用户、目录、依赖、初始化和参数优化。

📖

Markdown 知识库

内置树形组织笔记，Markdown 编辑预览，Lucene + IK Analyzer 中文全文搜索，支持图片粘贴与截图。

🤖

AI 辅助

内置 AI 问答面板，支持 DeepSeek、Qwen 等大模型，记忆开关灵活控制上下文，辅助 SQL 编写与问题排查。

🎨

主题与国际化

深色 / 浅色 Cupertino 主题，支持简体中文、繁体中文、英文界面切换。自定义无边框窗口，统一样式覆盖全部控件。

数据库适配

多数据库方言支持

通过方言模块扩展，不同数据库共享通用能力，同时可按自身特性提供专门的管理入口。

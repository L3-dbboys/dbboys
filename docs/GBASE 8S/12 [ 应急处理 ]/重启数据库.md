1、使用操作系统【gbasedbt】用户通过ssh或telnet登陆数据库服务器，执行以下命令关闭数据库，正常关闭无输出：
```
onmode -ky
```
2、使用【gbasedbt】用户执行以下命令启动数据库：
```
oninit -v
```

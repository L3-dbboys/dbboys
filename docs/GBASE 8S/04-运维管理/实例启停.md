### 启动数据库实例 
使用oninit -v启动数据库实例
```
oninit -v
```
最后为以下输出表示数据库已正常启动
```
Verbose output complete: mode = 5
```
如数据库未能正常启动，参考应急处理的数据库无法启动章节处理。


### 关闭数据库实例 
使用onmode -ky命令关闭数据库实例
```
onmode -ky
```
使用onstat -m检查数据库日志出现以下输出，表示数据库已正常关闭
```
00:03:39  GBase 8s Database Server Stopped.
```
如onmode -ky长时间无法关闭数据库，参考应急处理的数据库无法关闭章节处理。

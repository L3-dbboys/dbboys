### 说明
要移动数据文件路径，可将数据文件移动到目标路径，并在原路径创建连接指向新路径。要确认当前数据文件路径，可使用【onstat -d】查看。  


### 示例1-移动数据文件目录
将数据文件路径从系统盘/opt/gbase/dbs移动到存储盘/data  
关闭数据库，【gbasedbt】用户执行：
```
onmode -ky
```
移动数据文件目录到目标文件夹，【root】用户执行：
```
mv /opt/gbase/dbs /data
```
创建链接，【root】用户执行：
```
ln -s /data/dbs /opt/gbase/dbs
```
启动数据库，【gbasedbt】用户执行：
```
oninit -v
```

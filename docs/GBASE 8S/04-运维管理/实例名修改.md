### 查看当前实例名
```
env |grep GBASEDBTSERVER
```
### 修改实例名
关闭实例
```
onmode -ky
```
编辑实例环境变量文件并生效，修改GBASEDBTSERVER环境变量为新的实例名：vi .bash_profile
```
export GBASEDBTSERVER=gbase01_new
source .bash_profile
```
修改ONCONFIG配置文件中的实例名DBSERVERNAME为新的实例名：vi $GBASEDBTDIR/etc/$ONCONFIG
```
DBSERVERNAME gbase01_new
```
修改sqlhosts配置文件中的实例名为新的实例名：vi $GBASEDBTSQLHOSTS
```
gbase01_new  onsoctcp   *  9088
```
重启数据库实例【[实例启停](实例启停.md)】

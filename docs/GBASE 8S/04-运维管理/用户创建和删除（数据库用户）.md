### 创建用户
使用【gbasedbt】连接到实例
```
dbaccess - -
```
执行创建用户命令
```
CREATE USER user1 WITH PASSWORD "GBase123"; 
```
如使用的是数据库管理工具连接到数据库，需要使用gbasedbt用户连接到任意数据库，在sql界面执行创建用户命令即可。

### 授权
用户创建好以后，需要给该用户授予相应的权限才能连接数据库，参考以下示例。
授予【testdb】的【connect】权限，使用【gbasedbt】用户执行：
```
echo "grant connect to user1" | dbaccess testdb -
```
授予【testdb】的【dba】权限，使用【gbasedbt】用户执行：
```
echo "grant dba to user1" | dbaccess testdb -
```
如使用的是数据库管理工具连接到数据库，需要使用gbasedbt用户连接到需要授权的库，在sql界面执行授权语句即可。
### 测试用户可用性
连接实例
```
dbaccess - -
```
使用指定用户连接指定库，如以下使用user1连接gbasedb库
```
connect to 'gbasedb' user 'user1';
```
输入密码验证密码和权限
```
ENTER PASSWORD:
```
参考输出
```
Connected.   #连接成功
387: No connect permission.  #无连接权限，授权未成功
951: Incorrect password or user user1@localhost[localhost.localdomain] is not known on the database server.  #密码错误
```


### 删除用户
使用【gbasedbt】用户连接到实例
```
dbaccess - -
```
删除用户
```
drop user user1;
```
如使用的是数据库管理工具连接到数据库，需要使用gbasedbt用户连接到任意数据库，在sql界面执行删除用户命令即可。

### 检查系服务器配置
```
onstat -g osi
```
### 检查安装目录
```
env |grep GBASEDBTDIR
```
### 检查实例名
```
env |grep GBASEDBTSERVER
```
### 检查配置文件
```
env |grep ONCONFIG   #该文件位于$GBASEDBTDIR/etc目录，如无此环境变量，默认为onconfig
```
### 检查监听IP端口
```
cat $GBASEDBTSQLHOSTS   #离线，如无此环境变量，则默认为$GBASEDBTDIR/etc/sqlhosts
onstat -g ntt |grep soctcp   #在线
```
### 查看内存占用
```
onstat -  #最后一列显示数据库分配总内存
```
### 查看数据库进程数据量配置
```
onstat -c |grep VPCLASS
```
### 查看数据库空间使用
```
onstat -d
```

### 系统用户修改密码
如果使用的系统用连接数据库，使用【root】修改系统用户密码即可，如：
```
passwd gbasedbt;
```
### 数据库用户修改密码
如果使用数据库用户连接数据库，【gbasedbt】用户连接到任意数据库执行以下命令修改用户密码：
```
dbaccess - -<<!
alter user user1 modify password '8S*P)0Od@.&';
!
```

### 所有库总行数及总容量
```sql
SELECT
st.dbsname databasename,
sum(sin.ti_nrows) nrows,
round(sum(sin.ti_nptotal*sd.pagesize)/1024/1024/1024,2)||'GB' total_size,
round(sum(sin.ti_npused*sd.pagesize)/1024/1024/1024,2)||'GB' used_size
from
sysmaster:systabnames st JOIN sysmaster:systabinfo sin ON  st.partnum=sin.ti_partnum
JOIN sysmaster:sysdbspaces sd ON sd.dbsnum = trunc(st.partnum/1048576)
--WHERE st.dbsname='testdb' --如统计某个库在此写库名
GROUP BY 1
ORDER BY 3 DESC;
```
输出示例：
```
databasename  testdb
nrows         10319
total_size    0.01GB
used_size     0.01GB

databasename  rootdbs
nrows         492
total_size    0.01GB
used_size     0.01GB
```

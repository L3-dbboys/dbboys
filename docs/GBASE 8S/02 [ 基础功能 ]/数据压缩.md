评估压缩：
```sql
execute function sysadmin:task("table estimate_compression","t");
```
创建压缩字典，不压缩现有数据，压缩增量数据：
```sql
execute function sysadmin:task("table create_dictionary","t");
```
查看压缩字典：
```sql
select * from sysmaster:syscompdicts;
```
启用压缩：
```sql
execute function sysadmin:task("enable compression");
```
压缩表：
```sql
execute function sysadmin:task("table  compress","t")
```

### 创建系统用户
GBase 8s 默认使用系统用户连接数据库，要创建数据库用户，在操作系统创建系统用户即可。
如以下示例创建名为user1的数据库用户,【root】用户执行：
```
useradd user1
passwd user1
```
### 创建数据库用户
如不希望使用系统用户，直接创建数据库用户，需设置好用户映射后创建，按以下步骤创建：

1、开启用户映射，【root】用户执行:
```
mkdir /etc/gbasedbt
echo "USER:daemon" > /etc/gbasedbt/allowed.surrogates
```
2、设置相关参数，【gbasedbt】用户执行:
```
onmode -wf USERMAPPING='ADMIN' 
onmode -cache surrogates 
dbaccess sysuser - <<!
CREATE DEFAULT USER WITH PROPERTIES USER 'daemon'; 
!
```

3、创建用户：
```
dbaccess sysuser -<<!
CREATE USER user1 WITH PASSWORD "GBase123"; 
!
```

### 给用户授权
用户创建好以后，需要给该用户授予相应的权限才能连接数据库，参考以下示例。
授予【testdb】的【connect】权限，使用【gbasedbt】用户执行：
```
echo "grant connect to user1" | dbaccess testdb -
```
授予【testdb】的【dba】权限，使用【gbasedbt】用户执行：
```
echo "grant dba to user1" | dbaccess testdb -
```

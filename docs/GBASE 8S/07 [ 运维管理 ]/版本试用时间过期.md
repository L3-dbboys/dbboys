试用版使用时间为1年，超过该时间后，数据库保持正常运行，管理命令无法使用。
使用以下步骤替换试用版为正式版本或重装试用版，以下步骤默认数据库安装路径为【/opt/gbase】，如安装路径为其他值，操作时替换为实际值。

1、关闭数据库（gbasedbt用户执行）
```
onmode -ky
```

2、正式版安装到新目录（root用户执行）
```
tar -xvf GBase8s*.tar
./ids_install -i silent -DLICENSE_ACCEPTED=TRUE -DUSER_INSTALL_DIR=/opt/gbase_new
```
3、复制配置文件（root用户执行）
```
/bin/cp -p /opt/gbase/etc/onconfig*  /opt/gbase_new/etc/
/bin/cp -p /opt/gbase/etc/sqlhosts*  /opt/gbase_new/etc/
```
4、替换原有目录（root用户执行）
```
mv /opt/gbase /opt/gbase_old
mv /opt/gbase_new /opt/gbase
```
5、启动数据库（gbasedbt用户执行）
```
oninit -v
```

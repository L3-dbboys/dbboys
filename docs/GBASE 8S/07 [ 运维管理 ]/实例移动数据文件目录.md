以下示例将数据文件路径从系统盘/opt/gbase/gbaseserver_dbs移动到存储盘/data
要确认数据文件路径，可使用【onstat -d】查看。

1、关闭数据库，【gbasedbt】用户执行：
```
onmode -ky
```
2、移动数据文件目录到目标文件夹，【root】用户执行：
```
mv /opt/gbase/gbaseserver_dbs /data
```
3、创建链接，【root】用户执行：
```
ln -s /data/gbaseserver_dbs /opt/gbase/gbaseserver_dbs
```
4、启动数据库，【gbasedbt】用户执行：
```
oninit -v
```

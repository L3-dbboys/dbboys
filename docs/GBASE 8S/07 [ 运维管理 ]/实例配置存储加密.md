## 加密范围
支持用户密码复杂度、超时、尝试次数  
存储加密  
备份加密  

## 相关配置文件
```
cp $GBASEDBTDIR/etc/secswitch.std $GBASEDBTDIR/etc/secswitch.$GBASEDBTSERVER
```
修改参数
```
SECURITY_AUTHENTICATION=7
SECURITY_STORAGE_ENCRYPTION=1
SECURITY_BACKUP_RESTORE=1
```
修改配置文件securityconfig.std
```
ENCRYPTION_TYPE=SOFT
ENCRYPTION_TYPE=SOFT
```
初始化实例
```
oninit -ivy
```
验证存储加密
```
dd if=/data/gbase/rootdbschk001 of=1 count=1 bs=2k
hexdump -C 1
```
显示已没有明文

验证用户加密
```
onmode -wf USERMAPPING='ADMIN' 
onmode -cache surrogates 
dbaccess sysuser - <<!
CREATE DEFAULT USER WITH PROPERTIES USER 'daemon'; 
!
dbaccess sysuser -<<!
CREATE USER testuser WITH PASSWORD "12345678"; 
!
```
报错：961: The password is too simple.
